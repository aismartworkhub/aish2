목적
상품을 생성합니다.
URL
https://{기본 URL}/api/public/v1/products
HTTP 메서드
POST
헤더
Authorization: Bearer {api_key}
Accept: application/json
Content-Type: application/json 또는 multipart/form-data
본문
JSON 또는 multipart/form-data로 전송합니다.
필수
title (string): 상품명
category_id (integer): 카테고리 ID
featured_image (file|string): 메인 이미지 (파일, URL, data URI 가능)
선택
description_html (string): 상품 상세 HTML
status (string): pending | publish (기본값: pending)
base_price (number): 기본가 (기본값: 0)
sale_price (number): 할인가
is_on_sale (boolean): 할인 여부
is_free (boolean): 무료 여부
requires_shipping (boolean): 배송 사용 여부 (기본값: true)
is_shipping_free (boolean): 배송비 무료 여부 (기본값: true)
shipping_price (number): 배송비
is_over_50000_free (boolean): 5만원 이상 무료 여부
is_extra_shipping_fee (boolean): 추가 배송비 여부
is_purchase_agency (boolean): 구매대행 여부
tax_type (string): tax (과세) | taxfree (면세) | smallbiz (영세)
options (array): 옵션명 배열 (최대 2개)예: ["색상", "사이즈"]
variants (array): 옵션 조합 배열 (options 사용 시 필요)option1 (string)
option2 (string)
price (number)
stock (integer)
sku (string)
options_order (string): alphabetical | lowPrice | highPrice
no_option_stock (integer): 옵션 없는 상품의 재고
images (array): 추가 이미지 (파일, URL, data URI 가능)
thumbnail_link (string): 영상 썸네일 링크
installment_month (integer): 할부 개월 수 0~36
allow_link (string): 비공개 링크 키
