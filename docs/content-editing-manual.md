# AISH 콘텐츠·이미지 편집 매뉴얼

관리자(superadmin/admin)가 **페이지와 섹션의 이미지·내용**을 어디서, 어떻게 바꾸는지 정리한 문서입니다.
모든 편집은 `https://aish.co.kr/admin/...` 에서 하며, 저장 후 공개 페이지를 **강제 새로고침(Cmd/Ctrl+Shift+R)** 하면 반영됩니다. (캐시로 최대 수십 초 지연될 수 있음)

> ⚠️ 이 사이트는 파일 업로드 저장소를 쓰지 않습니다. **이미지는 모두 "URL"로 넣습니다.** (아래 [이미지 넣는 법](#5-이미지-넣는-법-공통) 참고)

---

## 1. 어디서 무엇을 수정하나 (한눈에)

| 수정하려는 것 | 메뉴 (사이드바) | 비고 |
|---|---|---|
| **메인 섹션 표시/순서/제목/여백** | 사이트 관리 → **메인 페이지 편집** | 신규. 섹션 끄기·순서·헤딩 |
| 메인 **히어로**(큰 배경 이미지·문구·버튼) | 사이트 설정 → **히어로 섹션** | 슬라이드 여러 장 |
| 숫자 실적(수강생 수 등) | 사이트 설정 → **실적 수치** | |
| 상단/하단 CTA 버튼 문구·링크 | 사이트 설정 → **CTA 설정** | |
| 상단 **D-Day 배너** | 사이트 설정 → **배너 관리** | |
| 홈 디자인 템플릿 | 사이트 설정 → **홈 테마** | 기본형/커뮤니티형/모던형 |
| 푸터 사업자 정보 | 사이트 설정 → **사업자 정보** | 법적 고지 |
| 페이지 히어로·소개 카드(이미지 포함) | 사이트 관리 → **페이지 관리** | 홈·소개 등 탭별 |
| 강사(프로필 이미지·소개) | 교육 운영 → **강사 관리** | 메인 "AI실전마스터" 섹션 |
| 교육 프로그램 | 교육 운영 → **프로그램 관리** | 메인 "Program" 섹션 |
| 스마트워크톤 / 행사 | 교육 운영 → **스마트워크톤 / 일반 행사** | |
| 후기 | (후기 관리) | 메인 "Review" 섹션 |
| 갤러리(드라이브 일괄 가져오기) | 콘텐츠 관리 → 갤러리 | |
| 공지/게시글 | 게시판 관련 메뉴 | 메인 "NewsRoom" |
| AI 상담 지식 | 사이트 설정 → **AI 지식** | 상담창 답변 근거 |

> 현재 라이브 홈 템플릿은 **커뮤니티형**입니다. 아래 섹션 설명은 커뮤니티형 기준입니다.

## 2. 메인 페이지 — 섹션 표시·순서·제목·여백

**메뉴: 사이트 관리 → 메인 페이지 편집** (`/admin/home-layout`)

메인 화면을 구성하는 큰 덩어리(섹션)들을 한 화면에서 제어합니다.

1. 각 섹션 카드 오른쪽 **표시/숨김** 토글 → 켜고 끄기
2. 왼쪽 **▲▼** 버튼 → 위아래 순서 바꾸기
3. 일부 섹션은 **제목·설명**을 직접 입력 (예: 교육 프로그램, AI실전마스터, 회원 혜택, Insight, 후기, 이벤트)
4. **상단/하단 여백(px)** → 비워두면 기본값, 숫자를 넣으면 그만큼 간격 조정
5. 맨 아래 **저장하기**

> - 여기서 바꾸는 건 섹션의 **틀**(표시·순서·헤딩·간격)입니다.
> - 섹션 **안의 콘텐츠**(강사 목록, 프로그램, 후기 등)는 각 콘텐츠 메뉴에서 바꿉니다. (1번 표 참고)
> - "홈 테마"를 바꾸면 이 편집기도 그 템플릿의 섹션 목록으로 자동 전환됩니다.

---

## 3. 메인 히어로 (상단 큰 배경 이미지)

**메뉴: 사이트 설정 → 히어로 섹션**

- 슬라이드를 여러 장 등록할 수 있고, **활성** 상태인 것만 자동 회전 노출됩니다.
- 각 슬라이드: **이미지 URL**, 제목, 부제목, 버튼 문구·링크.
- 제목/부제목은 **줄바꿈(Enter)** 이 그대로 반영됩니다.
- 이미지가 안 보이면 → [이미지 넣는 법](#5-이미지-넣는-법-공통) 확인.

---

## 4. 페이지별 내용 (히어로·섹션 제목·카드)

**메뉴: 사이트 관리 → 페이지 관리** (`/admin/pages`)

상단 탭(홈·소개·교육 프로그램 등)에서 페이지를 고른 뒤:

- **페이지 히어로**: 이미지 URL + 대제목 + 부제목
- **섹션 설명**: 일부 항목은 HTML 입력 가능
- **카드(이미지 포함)**: "교육 카드", "전문(Specialty) 카드" 등 — 카드마다 **이미지 URL** + 제목 + 설명. 카드 추가/삭제/순서 변경 가능.

> 참고(커뮤니티형 홈 기준):
> - 홈의 **"Specialty 카드"**(체계적 교육/실무 중심/커뮤니티 이미지 3장)는 여기 **홈 탭 → Specialty 카드**에서 바꿉니다.
> - 홈의 **섹션 제목/설명**(예: "AI실전마스터", "Member Benefits")은 이제 [메인 페이지 편집](#2-메인-페이지--섹션-표시순서제목여백)에서 바꿉니다. (페이지 관리의 홈 섹션 제목은 홈에 직접 반영되지 않습니다.)
> - **"회원 혜택(Member Benefits)" 6개 항목**은 코드에 고정돼 있어 제목/설명만 [메인 페이지 편집]에서 바꿀 수 있고, 항목 자체 변경은 개발 반영이 필요합니다.

## 5. 이미지 넣는 법 (공통)

이 사이트는 이미지를 **URL**로만 받습니다. 세 가지 방법이 있습니다.

### ① 구글 드라이브 (권장)
1. 드라이브에 이미지 업로드
2. 파일 우클릭 → **공유** → **"링크가 있는 모든 사용자"(보기)** 로 변경 ← **필수**
3. **링크 복사** → 그 링크를 이미지 URL 칸에 그대로 붙여넣기

아래 형식 모두 자동 인식됩니다(시스템이 표시용 주소로 변환):
- `https://drive.google.com/file/d/FILE_ID/view?...`
- `https://drive.google.com/open?id=FILE_ID`
- `https://drive.google.com/uc?id=FILE_ID`

### ② 외부 직접 이미지 주소
`https://...` 로 끝이 `.jpg/.png/.webp` 등인 **직접 이미지 URL**도 됩니다. (그 사이트가 외부 표시를 막지 않아야 함)

### ③ 사이트 기본 이미지
`/images/defaults/...` 같은 내장 경로도 사용 가능 (예: `/images/defaults/hero-main.jpg`).

### 입력 후 미리보기
편집 화면에 미리보기가 뜹니다. **"미리보기를 불러올 수 없습니다"** 가 보이면 대개:
- 드라이브 파일이 **"링크가 있는 모든 사용자"** 공개가 아님 → 공유 설정 다시 확인
- URL 오타 / 비공개 외부 이미지

## 6. 콘텐츠 항목(섹션 내부) 수정

메인 섹션 **안에 들어가는 실제 내용**은 각 전용 메뉴에서 관리합니다.

| 메인 섹션 | 내용 출처(메뉴) |
|---|---|
| AI실전마스터 | **강사 관리** (이름·직함·프로필 이미지·소개·전문분야) |
| 교육 프로그램 | **프로그램 관리** (외부 클래스 연동 시 자동) |
| 스마트워크톤 | **스마트워크톤** (포스터 이미지·일정·참가현황) |
| 행사/이벤트 | **일반 행사** (썸네일 이미지·일정·태그) |
| 후기(Review) | 후기 관리 (별점·내용·작성자) |
| NewsRoom | 공지 게시글 |
| Insight | 콘텐츠 허브(미디어)·영상 |

이미지가 들어가는 항목(강사 프로필, 행사 썸네일, 워크톤 포스터 등)도 모두 **URL 방식**입니다. [5번](#5-이미지-넣는-법-공통)과 동일.

> **갤러리 일괄 가져오기**: 콘텐츠 관리 → 갤러리에서, 공개 드라이브 **폴더 ID**를 넣으면 폴더 안 이미지를 한 번에 등록할 수 있습니다.

---

## 7. 저장·반영 규칙

- 각 화면의 **저장하기**를 눌러야 반영됩니다.
- 공개 페이지는 성능을 위해 **약 30초 캐시**가 있습니다. 바로 확인하려면 **강제 새로고침**(Cmd/Ctrl+Shift+R).
- 모바일은 화면을 아래로 당겨(풀투리프레시) 갱신할 수 있습니다.

## 8. 자주 묻는 문제

| 증상 | 원인·해결 |
|---|---|
| 이미지가 안 보임 / "미리보기 불러올 수 없음" | 드라이브 파일을 **"링크가 있는 모든 사용자"** 로 공개했는지 확인. 비공개·삭제·오타 점검. |
| 저장했는데 공개 화면이 그대로 | **강제 새로고침**(캐시). 시크릿 창에서도 확인. |
| 섹션 제목을 페이지 관리에서 바꿨는데 홈에 안 바뀜 | 홈 섹션 제목은 **메인 페이지 편집**에서 바꿉니다. |
| 섹션을 숨겼는데 다른 템플릿에선 그대로 | 표시/순서는 **현재 홈 템플릿** 기준입니다. 테마를 바꾸면 그 템플릿 설정을 따로 조정. |
| 회원 혜택 항목 내용을 바꾸고 싶음 | 항목 자체는 코드 고정 → 개발 반영 필요(제목/설명만 편집 가능). |
